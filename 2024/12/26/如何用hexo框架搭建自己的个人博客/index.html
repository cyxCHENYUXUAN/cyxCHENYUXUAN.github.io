如何用hexo框架搭建自己的个人博客

发表于 2024-12-26 分类于 Hexo 阅读次数：
本文字数： 1.6k 阅读时长 ≈ 1 分钟

一. 前置准备（安装git，node.js)

Git官网下载：https://git-scm.com/

Node.js官网下载：http://nodejs.cn/

注意安装好之后还要配置相应的环境变量（自行搜索教程）

二. 安装hexo框架

  1. 按住win+R，输入cmd进入命令行窗口，输入以下命令：

    npm install -g hexo

    等待安装完成，hexo文件夹会出现在nodejs/node_global/node_modules文件夹下，如下图：

  2. 新建一个文件夹myblogs用于存放博客

    这里我的个人博客已经搭建好了，展示如下：

    在你新建的空文件夹下直接点击鼠标右键->显示更多选项->git Bush here，如下图：

    为了能正确的使用git命令，你还需要输入你的GitHub用户名和关联邮箱，可以打开git bush here输入以下命令：

    git config --global user.name "Your Name"

    git config --global user.email "your_email@example.com"

    注意替换为自己的用户名和邮箱

  3. 初始化hexo

    在你新建的文件夹下右键打开git bush here，输入如下代码：

    hexo init

    等待初始化完成，你会发现刚才的文件夹已经有一些文件了，解释如下:

    ​ node_modules: 依赖包
    ​ public：存放生成的页面
    ​ scaffolds：生成文章的一些模板
    ​ source：用来存放你的文章
    ​ themes：主题
    ​ _config.yml: 博客的配置文件

    加下来查看能否正常启动，输入如下代码：

    hexo s

    复制http://localhost:4000/到浏览器打开，按住ctrl+c可以停止，效果如下：

    因为我已经对主题进行配置，所以不是原始界面

三. 将hexo部署到GitHub

  1. 创建个人仓库

    首先注册GitHub仓库（163邮箱，QQ邮箱，gmail均可），接着创建一个与你GitHub用户名相同的仓库，仓库名称为username.github.io，username替换为你的用户名

  2. 生成ssh密钥添加到GitHub

    进入家目录：C:\Users\86187，右键打开git bush here，输入如下代码：

    ssh-keygen -t rsa -C your_email@example.com

    注意替换为你注册GitHub时的邮箱

    之和敲三次回车，会发现C:\Users\86187下多了一个.ssh目录，点击打开，接着点击里面的id_rsa.pub，复制里面的内容后打开GitHub

    点击setting：

    点击SSH and GPG keys->New SSH key：

    Title名字随便起一个，将刚刚复制的SSH密钥粘贴到Key中，完成后点击Add SSH Key

  3. 进行部署

    打开myblogs下的_config.yml文件，这是站点配置文件，用VScode打开编辑，滑动鼠标滚轮到文件的最后，修改deploy中的内容，如下所示：

    获取repo的步骤如下：

    打开刚才新建的个人仓库，接着进行如下操作：

    打开新建的myblogs文件夹，先安装deploy-git插件，这样才能进行部署，打开git bush here，输入如下代码：

    npm install hexo-deployer-git --save

    执行完成之后继续按顺序执行如下命令：

    hexo clean #清除缓存文件db.json和已生成的静态文件public

    hexo g #生成网站静态文件到默认设置的public文件夹(hexo generate的缩写)

    hexo d #自动生成网站静态文件，并部署到设定的仓库(hexo deploy的缩写)

    注意：第一次输入hexo d时会让输入yes，之后就没有了

    回到GitHub个人仓库，按F5刷新页面，你会发现文件已经上传成功，之后就可以使用username.github.io访问个人博客了，记得将username改为你的GitHub用户名
